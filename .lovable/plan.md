

# 온보딩 가이드 툴팁 추가 계획

## 목표
첫 방문 사용자에게 "데모 데이터로 시작하기" 버튼을 안내하는 툴팁을 자동으로 표시하여 진입 장벽을 낮춤.

## 동작 방식
- 첫 방문 시 데모 버튼 위에 말풍선 형태의 가이드 툴팁이 자동으로 나타남
- "데이터가 없으신가요? 데모 데이터로 먼저 체험해보세요!" 같은 안내 문구 표시
- `localStorage`에 `finpilot-onboarded` 플래그를 저장하여 재방문 시에는 표시하지 않음
- 툴팁에 닫기(X) 버튼 포함, 또는 데모 버튼 클릭 시 자동으로 사라짐
- 부드러운 bounce/pulse 애니메이션으로 시선 유도

## 구현 방식

두 가지 접근이 가능합니다:

### A. 커스텀 말풍선 (추천)
- 별도 라이브러리 없이 데모 버튼 바로 위에 absolute position으로 말풍선 div 렌더링
- 화살표(▼) 포함한 간단한 CSS 말풍선
- 장점: 가볍고 디자인 자유도 높음

### B. Radix Popover 활용
- 이미 설치된 Radix Popover를 `defaultOpen`으로 사용
- 장점: 기존 컴포넌트 재사용, 위치 자동 계산
