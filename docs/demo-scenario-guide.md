# 3분 데모 시연 가이드 (Demo Scenario Guide)

면접·포트폴리오 리뷰에서 화면을 공유하며 시연할 때의 스크립트입니다.
내장 가상 데이터(2025.01~2026.06, 18개월 + 본예산/수정예산)에는 **의도적으로 심어둔 시나리오**가 있어, 아래 동선대로 가면 도구의 핵심이 3분 안에 전달됩니다.

> 사전 준비: 새 브라우저 탭 → 접속 → "가상 데이터로 시작하기" 클릭 → 기준월 2026년 6월 확인. (Gemini 키를 미리 넣어두면 코멘트가 자동 생성되고, 없어도 규칙 기반 초안으로 시연 가능 — 오히려 "AI 없이도 돌아간다"는 포인트로 쓸 수 있음)

---

## 0:00–0:30 | 시작 화면 → 월간 현황

**말하기**: "ERP에서 내려받은 엑셀을 양식 그대로 올리는 구조인데, 오늘은 내장 가상 데이터로 보겠습니다. 18개월 치 전표와 예산 2개 버전이 들어있습니다."

- 월간 현황 상단: 요약 카드 4그룹 — "총비용, 전월 대비, 예산 대비, 확인 필요 항목. 여기까지가 30초 요약입니다."
- 12개월 실적 vs 예산 차트: "막대가 실적, 점선이 예산 — 최근 몇 달 예산선을 넘는 패턴이 보입니다."

## 0:30–1:10 | 검토 포인트 → 동인 분해 (핵심 장면 ①)

**말하기**: "이 도구의 핵심은 '어디를 봐야 하는지'를 먼저 알려주는 겁니다."

- 검토 포인트 블록에서 **지급수수료** 항목을 가리키기 — 거래처 기여도가 함께 표시됨.
- "지급수수료가 늘었다, 까지는 엑셀 피벗도 알려줍니다. 이 도구는 **증가분의 몇 %가 어느 거래처인지**까지 분해합니다 — 가상 데이터에서는 PG사 결제대행 수수료가 주요 동인으로 잡히도록 되어 있고, 이런 매출 연동 비용은 초과 자체보다 요율·연동성 확인이 포인트라는 것까지 이어집니다."
- 워터폴 차트: "전월 대비 총액 변화를 계정별로 분해한 그림 — 무엇이 밀어올렸는지 한눈에."

## 1:10–1:50 | 마감 검토 — 사유 초안과 이상 전표 (핵심 장면 ②)

**말하기**: "실무에서 제일 번거롭던 두 가지가 여기 있습니다."

- 마감 검토 → 변동사유: "검토 대상마다 **사유 초안이 이미 채워져** 있습니다. 거래처 기여도와 적요 기반의 규칙 초안이고, 사람은 읽고 고치고 '확정'만 누릅니다. 확정된 사유는 월간 리포트에 그대로 올라갑니다."
- 전표·경비 검토 전환: "제가 실제로 하던 경비 검토를 규칙 7종으로 옮긴 겁니다." — **2026년 5월로 이동해 중복 결제 의심(접대비 동일 거래처·동일 금액)**, 6월의 **분할 결제 의심(소모품비 같은 날 다건)**, VAT 불일치 검출을 보여주기. "검출은 오류 확정이 아니라 확인 목록이고, '소명 필요' 처리한 건은 보고서 내부통제 요약으로 올라갑니다."

## 1:50–2:30 | 월간 리포트 (핵심 장면 ③)

**말하기**: "최종 산출물은 결국 보고서입니다."

- 월간 리포트: 핵심 요약 + 집행률 게이지 → 변동 요인 표(기여도·동인 후보·성격 열 강조) → 예산 차이(초과 표에 사유 후보가 붙어있음) → **확인 필요 항목**("모르는 건 모른다고 분리하는 게 보고의 신뢰라고 생각합니다") → 보고용 코멘트 초안("그대로 복사해 쓰되, '검토 후 사용' 라벨이 원칙입니다").
- 인쇄 버튼: "이 화면 그대로 PDF가 되어 보고 초안이 됩니다."

## 2:30–3:00 | 마무리 — 설계 원칙 한 줄

**말하기**: "정리하면 — 계산과 선별은 규칙 엔진이, 문장 초안은 AI가, **확정과 책임은 사람이** 갖는 구조입니다. 그리고 직접 해본 업무(경비 검토, 증감 분석)는 깊게, 경험 없는 영역(예산 편성, Forecast)은 범위 밖으로 명시한 것이 이 포트폴리오의 원칙입니다."

## 예상 돌발 상황 대응

- **코멘트가 "호출 한도 대기 중"**: "무료 API 한도라 잠시 대기 후 자동 재시도됩니다. 그 사이에도 규칙 기반 초안과 모든 수치는 그대로 동작합니다" — 오히려 장애 대응 설계를 보여주는 기회.
- **"이 숫자 맞아요?" 질문**: 상세 분석 탭 → 해당 계정 클릭 → 거래처 구성·월별 추이로 내려가 원자료 경로를 보여주기. "모든 요약 숫자는 전표까지 추적 가능합니다."
- **"실데이터로도 되나요?"**: 데이터 관리 → 업로드 → 컬럼 매핑 화면을 잠깐 보여주며 "양식이 달라도 여기서 연결하고, 한 번 연결한 양식은 프로파일로 저장됩니다."

## 시연 금지 사항

- 실제 회사 데이터로 시연하지 않기 (보안·비밀유지)
- AI 코멘트를 "정답"처럼 읽지 않기 — 반드시 "초안이고 검토가 남아있다"는 말과 함께
- 가상 데이터의 수치를 실제 경험 수치처럼 말하지 않기 ("가상 데이터 기준으로는~"으로 시작)
